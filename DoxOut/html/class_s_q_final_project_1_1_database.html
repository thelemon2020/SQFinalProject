Detailed Description

The purpose of this class is to act as an abstraction for the DatabaseInteraction class. It takes arguments passed in from the various Window classes, parses them and then passes them onto the various DatabaseInteraction methods. All error handling is handled by the DatabaseInteraction methods, which just pass back null if there is an error. As the returns from all Database methods are strings or List<string>, a null is easily parsed as a failed query.
