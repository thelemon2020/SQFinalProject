A constructor for the ContractDetails class.

Details The constructor for the ContractDetails class, it takes an assortment of parameters which are to be gathered from the contract marketplace database. A number of parameters of the class are not set until a carrier for the contract is selected from the Carrier database.

Parameters
-name - string - The customer's name
-job - int - The job type identified by an integer
-quant - int - the quantity of pallets to deliver, 0 if the delivery is FTL
-origin - string - The origin city
-dest - string - The destination city
-van - int - The van type identified by an integer
